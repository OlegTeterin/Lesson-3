Введение в контроль версий
Раздел 1
1.1. Коммнады в git
git init - создаёт локальный репозтитарий
git add . - добавляет файловую версионность в репозтитарий
git commit -m "Text ....." - комментарии к изменениям
git diff - показывает не записанные изменения 
git log - вывод всех commit
git log --oneline - выводит все commit в сокращенном варианте
git log --all --oneline - выводит все commit в сокращенном варианте
git checkout master - выводит на последнюю версию изменений
git branch - показывает ветки
